# Moomoo Warrior
A thrilling 2D survival game. Join fast-paced multiplayer battles and dominate the server!

## Play Now
[Moomoo Warrior](https://moomoowarrior.mooo.com)

## Description
MooMoo Warrior is a fan-made version of MooMoo.io, created due to the growing concerns over hackers and the lack of updates from the original developers. Hope this game can bring a fresh, secure, and continuously evolving experience for all the fans of the original game.
